# SIGEH-PHP Hotel Administration

SIGEH-PHP is a Laravel hotel admin system for room operations, reservations, guests, billing, payments, services, reporting, users, roles, permissions, and audit trails.

## Key Features

- Responsive hotel admin dashboard
- Room, guest, booking, reservation, invoice, payment, and service management
- PDF invoices and report exports
- Excel report exports
- Operational, financial, and statistical reports
- Audit logging for authentication and data changes
- CSRF protection, input sanitization, encrypted sessions, and permission middleware
